## Design System Reference

All designs follow **Besty Design Standards** from `docs/#DESIGN_SYSTEM.md`:

- **Colors**: Purple (#8B5CF6) to Indigo (#6366F1) gradients
- **Glass**: `bg-white/80 dark:bg-white/10 backdrop-blur-md`

- **Rounded**: Generous (`rounded-2xl`, `rounded-3xl`)
- **Hover**: `-translate-y-1`, `scale-105`, `duration-200`
- **Typography**: Bold gradients for headings
- **Touch Targets**: Minimum 44px

## Accessibility Checklist

- [ ] All interactive elements min 44x44px
- [ ] Keyboard navigation (Tab, Enter, Arrows)
- [ ] Screen reader labels (aria-label)
- [ ] Color contrast WCAG AA (4.5:1 text)
- [ ] Focus indicators (ring-2 ring-purple-500)
- [ ] Alt text for all icons/images
- [ ] Skip to content link
- [ ] Error messages clear and helpful
